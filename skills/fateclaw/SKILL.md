# FateClaw (玄学大狮) 🧙‍♂️

_AI-Powered Chinese Divination & Destiny Analysis_

基于 AI 大模型的智能玄学助手，为你提供八字分析、易经占卜、运势预测等服务。

---

## 🎯 核心功能

### 1. AI 八字分析 🔮
根据出生时间，AI 生成专业的八字分析报告。

**使用方式：**
```
/八字 1990年5月15日14时 男
帮我看看八字，1995年8月20日早上7点出生，女
```

**AI 将分析：**
- 四柱八字排盘
- 五行强弱分析
- 性格特点解读
- 事业、财运、感情、健康建议
- 开运建议

---

### 2. AI 易经占卜 📿
心中默念所问之事，AI 为你起卦解读。

**使用方式：**
```
/占卜 今年创业能否成功？
/算卦 问姻缘
占卜一下这次考试能不能过
```

**AI 将提供：**
- 随机生成的卦象
- 卦辞爻辞解读
- 针对具体问题的建议
- 应期预测

---

### 3. 每日运势 📅
根据生肖和日期，生成个性化运势。

**使用方式：**
```
/运势 龙
我属马，看看今天运势
今日运势如何
```

**包含内容：**
- 整体运势评分
- 事业、财运、感情、健康分项
- 幸运方位、颜色、数字
- 宜忌事项

---

### 4. 风水建议 🏠
根据房间类型和朝向，提供 AI 风水建议。

**使用方式：**
```
/风水 办公室 朝南
卧室风水怎么布置
```

---

### 5. 姓名分析 📝
分析姓名笔画数理，解读吉凶。

**使用方式：**
```
/姓名 李明
帮我看看这个名字怎么样
```
